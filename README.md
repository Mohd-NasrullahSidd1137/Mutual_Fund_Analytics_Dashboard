# 📊 Mutual Fund Analytics Dashboard

## 📌 Project Overview

The **Mutual Fund Analytics Dashboard** is an interactive **Power BI** project designed to analyze mutual fund performance, investor behavior, portfolio allocation, and market trends. This dashboard enables users to explore key financial metrics, compare fund performance, and gain actionable insights through dynamic visualizations.

---

## 🎯 Objectives

* Analyze Mutual Fund performance across different fund houses.
* Monitor important KPIs such as AUM, NAV, Returns, Expense Ratio, and Sharpe Ratio.
* Understand investor demographics and investment behavior.
* Compare portfolio allocation across sectors and holdings.
* Evaluate fund performance using risk-return analysis and ratings.
